# Outputs del Proyecto

Los output incluyen el padrón de IIEE focalizadas y los costeos a nivel de DREUGEL y UE. Se incluyen costeos para TaRL y ConectaIdeas.

| Input | Output |
|-------|--------|
| output/tabla_resumen_1.xlsx | Presenta estadísticas descriptivas de la focalización | 
| output/padron | Presenta el padrón de IIEE seleccionadas para la focalización | 
| output/PxQ_DREUGEL | Presenta el costeo agregado a nivel de DREUGEL | 
| output/PxQ_UE | Presenta el costeo agregado a nivel de UE |
